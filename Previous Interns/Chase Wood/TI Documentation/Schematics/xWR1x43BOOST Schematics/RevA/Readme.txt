The design database provided includes the following contents
1) The "Gerber" folder includes the fabrication and assembly gerber files. These include all layer details.
2) The “Layout” folder includes the .brd layout file that can be viewed in the Cadence Allegro tool. It also has the layout version exported in pdf for easy viewing.
3) Stackup is the construction of the PCB layers used for the PCB manufacturing. 
4) The database.7z is a zip file with the schematic design database. This can be viewed using Candence ADW
