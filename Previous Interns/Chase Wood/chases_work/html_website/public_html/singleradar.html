Single mmWave Radar GUI

University of Texas at Dallas

Description:

  • This application includes functionality of the initial mmWaveStudio - LUA command GUI. In addition to these features, this application incorporates a utility for connecting and controlling TI mmWave radars using TI's CLI commands found in the mmWaveSDK, allowing this utility to run with or without mmWaveStudio running by selecting either the mmWaveStudio or CLI mode of operation.

  • This application also integrates HW trigger functionality for both of the modes of operation. When used with mmWaveStudio, the appropriate LUA command is sent to enable/disable HW trigger when selected. When used in the CLI mode of operation, one of the commands is adjusted when configuring the radar to reflect the chosen trigger mode.

  • mmWaveStudio radar firmware and LUA commands. (mmWaveStudio must be running for entire process)

  • mmWave SDK firmware for command line interface (CLI) operation. (mmWaveStudio closed for entire process)

  • Hardware trigger control supported for both operation modes.
