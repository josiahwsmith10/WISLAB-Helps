Dual mmWave Radar GUI

University of Texas at Dallas

Description:

  • This application supports the configuration and data capture of [up to] two mmWave radars using the CLI functionality of the mmWave SDK out-of-box firmware.

  • For dual-radar operation, this tool requires the following to be connected to a single PC:
    • Qty. 2 - mmWave radars (both 60GHz and 77GHz radars supported)
    • Qty. 2 - DCA1000EVM (data capture devices set to unique IP addresses)

  • Application supports two unique HW trigger signals supplied by a single MCU (ESP8266).
  • Utilizes command line interface for controlling radars and data capture cards.
  • Allows data capture using independent radars and data capture devices without the use of mmWaveStudio.
